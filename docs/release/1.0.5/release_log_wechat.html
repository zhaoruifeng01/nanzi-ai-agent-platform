🚀 南孜 AI 智能体平台 · v1.0.5 重磅发布

联邦查数加固 · 黄金报表 · SQL Server · 第三方用户同步

GitHub Repository： RandyChen1985/nanzi-ai-agent-platform

v1.0.5 是一次ChatBI 联邦查数深度加固、数据门户黄金报表、SQL Server 与第三方用户同步的大版本更新。联邦 WHERE 探查与空结果自动重试全面增强；Portal 黄金 SQL 报表支持持久化、分享与个人偏好；新增 SQL Server 全链路支持；第三方用户同步重构为「用户名映射 + 扩展 JSON 聚合」。

本次变更范围自 61681d9（含）至 099181d，共 49 个提交，涉及 199 个文件、约 21,800 行新增代码。

🚀 Key Features

01 🔗 ChatBI 联邦查数深度加固

● memory_join 计划修正、联邦 SQL repair 与降级留空机制。

● WHERE 探查接入 Schema；空结果筛选诊断与自动重试。

● 子查询门禁、平台重试预算与 JOIN 异常拦截。

02 📊 数据门户黄金报表体系

● Portal 保存/分享/运行黄金 SQL 报表，持久化至数据库。

● 个人偏好（收藏/置顶/运行历史）、放大浏览与表格视图切换。

03 🗄️ SQL Server 全链路支持

● 新增 SQL Server 数据源（ODBC）、元数据导入与本地只读查询。

● 修复 sqlglot 方言映射，ChatBI 可正常解析 T-SQL；Docker 镜像含 ODBC 驱动。

04 👥 第三方用户同步（重构）

● 以用户名为映射主键：不存在则新增，已存在则更新。

● 来源表多字段聚合为 JSON 写入 extra_data；支持定时同步。

05 🤖 子代理与 Office 文档工具

● 子代理委派链路优化；新增 Word/Excel 文档工具与生成文件下载。

06 ⚙️ 平台管理与运维

● 品牌个性化、知识库总开关、主助手动态专家清单。

● 智能体中心拖拽排序；Redis 诊断增强；行级权限重写完善。

● 数据门户 Redis 缓存 TTL 延长至 90 天。

⚠️ Breaking Changes

项目 说明
数据库 须执行 V82 ~ V87 六个增量脚本。
用户同步 改为用户名映射，旧 field_map.id 已废弃。
知识库 新增总开关，关闭后相关入口不可用。
Docker SQL Server 需含 ODBC 驱动的新镜像。
