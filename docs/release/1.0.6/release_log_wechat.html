🚀 南孜 AI 智能体平台 · v1.0.6 重磅发布

知识库运营看板 · Token额度预算 · 工作空间升级3.0 · 多渠道通知

GitHub Repository： RandyChen1985/nanzi-ai-agent-platform

v1.0.6 是一次以“知识库开发与管理、Token 额度计费、工作空间及多端消息通知”为核心的里程碑大版本迭代。平台全面支持知识库虚拟文件夹分类、卡片独立折叠、自建徽章标记、AI 元数据智能分析与双栏推荐问题；全新上线知识库运营统计看板与原档高亮物理定位预览；同时引入 Token 额度控制机制（支持 `/quota` 快捷指令），工作空间文件浏览全面升级（支持回收站与私有 sandbox 目录），并重构个人消息配置支持多端连通性测试。

本次变更范围自 d429e7f（含）至 64e40cf，共 87 个提交，涉及 161 个文件、约 25,364 行新增代码。

🚀 Key Features

01 📂 知识库开发与 RAG 管理体系深度加固

●文档虚拟目录分类：支持多级虚拟文件夹逻辑划分，文件树拖拽归类与重命名，并在行 hover 时增加防文字重叠自适应。

●反向成员权限分配：支持按角色或用户批量对指定数据集分配或解绑操作权限，限权用户将无法越界访问敏感数据集。

●AI 元数据分析与回填：编辑元数据弹窗集成大模型，一键对多文件命名趋势进行语义推断，智能提取中英文标签和描述备注。

●双栏推荐提问交互：知识库置顶预设提问与大模型动态预测提问以独立双栏优雅排版呈现，支持直接发送或铅笔 ✏️ 编辑后再发送。

●友好错误翻译拦截：`RagFlowClient` 自动截获并翻译底层的 UUID 形式报错（如权限拒绝、找不到数据集等），转换为具备操作指引的中文友好提示。

●检索优化与 Mermaid 生成：识别代词指代追问，自动短路跳过 RAG 检索并复用历史引用；优化 Mermaid 流程图生成提示词确保页面无错渲染。

02 📊 知识库运营统计看板与原档高亮

●运营看板统计看板：可视化统计看板正式上线，呈现高频使用排行、周检索趋势及缓存击中数据，刷新看板时自动触发 Redis→DB 实时物理落库同步。

●物理页码高亮滑出预览：引用的事实依据（Citations）支持直接定位到原始 PDF/Markdown 的物理页码，侧边一键滑出原档渲染页面并高亮匹配段落。

●埋点监控修复：彻底修复了部分极端场景下日志拦截器采集不到 Citation 引用元数据的 Bug，强化数据链路追踪。

03 🗂️ 工作空间文件浏览器升级 3.0

●多维文件防灾与隔离：工作空间浏览偏好、最近打开文件改为 Redis 按 `user_id` 物理隔离，新增用户私有 `uploads/sandbox` 目录。

●回收站与快捷导航：支持文件软删除逻辑并移入回收站暂存，提供“最近打开”和“快捷收藏导航”面板。

●右侧抽屉与编辑 API：重构为抽屉式文件管理器，修复画布渲染 blob URL 破图故障，支持远程文件流写入与预览 API。

04 💰 Token 额度控制与用量分析体系

●多维度策略拦截：后台可针对具体用户或角色设置 Token 预算策略，拦截超额请求，有效防范 API 被恶性刷量或脚本死循环空转。

●个人消耗详情看板：个人中心新增消耗统计仪表盘，数值统一转换为 `12.5 K` / `1.2 M` / `2.3 B` 等直观的数据紧凑格式展示。

●`/quota` 快捷命令：在对话输入框直接发送 `/quota`，主助手将实时返回您本月已分配配额、当前消耗量以及剩余的可用额度。

05 🔔 个人中心消息通知多渠道绑定

●一键多通道连通性测试：个人配置页支持统一管理并绑定钉钉群机器人、企业微信群机器人、SMTP 邮件服务，支持密码打星脱敏与连通性测试。

●智能体通知工具重构：系统预装的通知发送工具彻底重构为“自动读取当前对话用户的消息配置”逻辑，新增企业微信机器人群发送工具。

06 ⚙️ 智能路由与统一请求决策

●统一请求决策层重构：抽取 `request_decision.py`，合并判定 `RequestSource` 与 `RequestCapability`；优化 `resolve_turn_classification`，在启发式判定和语义映射阶段复用同一决策，消灭重复运算。

●网页悬浮挂件优化：系统悬浮助手集成引导页全面升级，支持 Widget 弹窗与 Embedded 嵌入等多模式调用。

●跨设备活动会话同步：使用 Redis 维护无痕或跨设备状态下的活动会话 ID，保证会话一致性。

⚠️ Breaking Changes

项目 说明
数据库 须执行 V88 ~ V93 六个增量脚本。
Token 限额 系统启用 Token 额度拦截机制，需正确配置 quota_policies 表。
通知密码 明文密码弃用，保存时自动加密传输与落库。
沙箱路径 临时 Sandbox 路径变更，严禁在外部工具内硬编码物理路径。
