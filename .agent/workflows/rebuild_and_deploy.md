# 一键重部署工作流 (Rebuild & Deploy Workflow)

该工作流用于在修改代码（前端或后端）后，快速完成编译、重启和状态检查。

## 📋 流程描述

1.  **编译前端**：进入 `frontend` 目录执行 `npm run build`。
2.  **清理环境**：检测并强制关闭占用 `8001` 端口的旧进程。
3.  **启动服务**：使用 `venv` 环境下的 `uvicorn` 在后台启动服务。
4.  **健康检查**：检查进程状态并输出启动日志的前几行。

## 🚀 如何执行

在项目根目录下运行以下命令：

```bash
./scripts/redeploy.sh
```

或者让 Agent 执行：

> "帮我重新部署一下"
> "运行重部署脚本"

## ⚙️ 脚本位置

- 脚本路径：`scripts/redeploy.sh`
- 日志路径：`server.log`
