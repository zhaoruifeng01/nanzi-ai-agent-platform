# 提案：统一对外对话接口雏形 (Unified Chat API V1 Prototype)

## 摘要
本提案旨在建立“云枢・智能体平台”的第一个正式对外接口版本（V1）。该接口将作为一个统一的 AI 通信入口，能够根据用户输入的语义自动路由到不同的处理模块（如 ChatBI、RAG 或通用对话）。

## 背景
目前的 AI 接口散落在私有门户（Portal）下，且功能单一。为了支撑第三方系统集成需求，我们需要一个标准化的、具备路由能力的入口。

## 目标
- **标准化**：采用类似 OpenAI 的接口契约，降低集成成本。
- **智能化路由**：基于已有的 `IntentService`，实现请求的自动分发。
- **扩展性**：支持简单的对话历史上下文。
- **安全隔离**：建立专用的 V1 API 路由空间。
